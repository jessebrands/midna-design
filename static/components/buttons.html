Buttons

Priority

Buttons are categorized by their importance, communicating intent to the user on the kind of action that will be taken by pressing one of them. Colored buttons should be used very sparingly.

Example

Disabled

To disable a button, prefer using the semantic HTML disabled attribute. For cases where this attribute is not available, the button--disabled CSS class can be used.

Example

Buttons with an icon

To further communicate intent to the user, you can opt to include a phosphor icon with the button. You can apply the button__icon CSS class to the icon to style it properly.
