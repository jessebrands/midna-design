Utilities

Helper classes

text-small for small text.

text-secondary for lighter text.

font-bold for 600 weight bold text.

font-light for 300 weight light text.

font-italic for italic font.

SCSS/SASS variables

$font-stack-base for the default font stack ‐ "Poppins", "Century Gothic", "Verdana", sans-serif.

$font-stack-mono for the monospace font stack ‐ "SFMono-Regular", Menlo, Monaco, Consolas, "Liberation Mono", "Courier New", Courier, monospace.

$font-size-large for 17px large text.

$font-size-regular for 16px regular text.

$font-size-small for 14px small text.
